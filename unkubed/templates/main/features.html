Unkubed capabilities

Features that map UI to kubectl

Every screen is designed to teach Kubernetes fundamentals through clear visuals, gradients inspired by the cluster nebula, and annotated screenshots.

Contextual dashboard

A single glance shows namespaces, workloads, services, and the exact kubectl get commands that populate them.

Template studio

Generate deployment, service, and config map YAML with smart defaults, preview the manifest, and see the matching apply workflow.

Pod health insights

Combine events, logs, and restart counts into gradient-highlighted diagnostics that suggest the next command or probe to inspect.

Command history

Unkubed stores the gradient-highlighted commands you triggered along with success status and timestamps for future study.

Safe cluster connections

Read-only flows grounded in your kubeconfig ensure clarity without risking a production cluster.

Sunset-aware theming

Automatic light-to-dark transitions follow local sunset times, keeping late-night study sessions comfortable.
